## Mô tả
Đây là dự án game Space Shooter 2D được phát triển trên Unity. Bài nộp này bao gồm các chức năng đã hoàn thành trong phần Demo Part 1.

## Các tính năng
- Thiết lập Player: Tàu vũ trụ với tỷ lệ hiển thị phù hợp.
- Di chuyển: Tàu di chuyển theo vị trí chuột trên màn hình.
- Hiệu ứng (VFX): Hiệu ứng lửa đuôi tàu nhấp nháy tự động bằng script.
- Cấu hình: Tối ưu cho độ phân giải 1920x1080 (Portrait).

## Công nghệ sử dụng
- Engine: Unity
- Ngôn ngữ: C#

## Hướng dẫn chạy
1. Clone repository này về máy.
2. Mở Unity Hub và Add thư mục project vào.
3. Mở Scene tên là "SpaceShooter" trong thư mục Assets/Scenes.
4. Nhấn nút Play để chạy thử.
